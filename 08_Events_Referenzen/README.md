# Events-Referenzen — Bilder von durchgeführten Events

Hier kommen Fotos von echten Events rein, die als Slider auf der Startseite angezeigt werden.

## Wie ablegen

Einfach Fotos direkt in diesen Ordner ziehen. Beliebige Dateinamen — ich benenne sie beim Einbau in die Site sauber um.

## Was sich gut macht

- **Außenaufnahmen vom Bus auf dem Event** (Wow-Effekt, Hingucker)
- **Gäste-Reaktionen** (Schlangen, Lächeln, Gruppen)
- **Innenraum mit Gästen** (Posing, Spaß)
- **Detail-Shots** (Fotostreifen, ausgedrucktes Foto in Hand)

## Optional — pro Bild eine kurze Beschreibung

Falls du Lust hast, kannst du eine kleine Datei `event-bezeichnung.txt` daneben legen mit:
- Anlass (Hochzeit / Firma / Stadtfest / Messe / Behörden)
- Datum
- Ort (nur Stadt, nicht Adresse — DSGVO)

Beispiel: „Hochzeit · April 2026 · Saarbrücken"

Aber kein Muss — Bilder allein reichen erstmal.
