## 6. Plugin-Stack WordPress (empfohlen)

| Aufgabe | Plugin |
|---|---|
| SEO | **Rank Math** (besser als Yoast für lokale Sites) oder **Yoast SEO** |
| Performance | **WP Rocket** oder kostenlos **W3 Total Cache** |
| Bilder | **ShortPixel** oder **Smush** (für WebP-Auslieferung) |
| Anfrageformular | **Contact Form 7** oder **WPForms** |
| DSGVO | **Borlabs Cookie** |
| Backups | **UpdraftPlus** |
| Schema.org | meist integriert in Rank Math/Yoast |

## 7. Strukturelle Cross-Linking-Strategie

**Auf fotobus.saarland:**
- Footer-Link „Weitere Fotoboxen ansehen → fotobox.saarland"
- Auf der „Preise"-Seite: „Klassische Fotobox günstiger? → fotobox.saarland"

**Auf fotobox.saarland:**
- Im bestehenden Bereich „Feuerwehrbus" einen prominenten Button „Eigene Webseite des Busses → fotobus.saarland"
- Im Footer/Header: „Unsere Marken" → beide Domains

> Vorteil: kein „Doppelcontent" (zwei verschiedene Schwerpunkte), aber gemeinsame Markenstärke. Suchmaschinen verstehen die Beziehung über Cross-Links.
