# Fotobus Saarland 🚐📸

Mobile Fotobox im umgebauten Feuerwehr-Bus – Eventfotografie für Hochzeiten, Firmenfeiern, Stadt- und Vereinsfeste im Saarland und Umgebung.

## Über das Projekt

Der **Fotobus Saarland** ist eine mobile Fotobox in einem liebevoll umgebauten ehemaligen Feuerwehrbus. Mit Glitzervorhang, individueller Beleuchtung und hochwertiger Technik sorgt der Bus für unvergessliche Erinnerungen bei jedem Event.
